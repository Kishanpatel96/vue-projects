# Music-App

## Description
> This is simple a Music application.

> In this application you can do play, pause, next and previous action on songs.

> This is Web and Mobile friendly.

## Languages Used
  - Vuejs (Version 2).
  - CSS
  - HTML
  
### Compiles and hot-reloads for development
```
npm run serve
```
